# API Gateway Demo

## Instalação

Instalar o servidor rabbitmq na máquina e instalar o python e as dependências

```
pyenv install 3.5.2
pip install pika
```

## Execução

Depois de iniciar o rabbitmq, rodar os serviços

```
# No primeiro terminal
python apigateway.py

# No segundo terminal
python auth.py
```
